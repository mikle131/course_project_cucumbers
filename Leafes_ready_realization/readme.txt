1. Ввести в командной строке pip install -r requirements. txt для установки необходимых для работа библиотек
2. Запустить программу командой python main.py
3. Дождаться сообщения Bot is ready. Теперь можно отправлять боту https://t.me/cucumber_diseases_bot фотографии,
   он их будет обрабатывать и отправлять 3 наиболее вероятных категории

P.S. Для тестирования другой версии реализации необходимо завершить работу текущей программы
